## நோக்கம் / Intent (Soul: WHY)

Agents need infrastructure, not just prompts. Agentic-kit provides the battle-tested primitives to move AI agent systems from demo to production: semantic memory, action dispatchers, circuit breakers, and reflection loops. Powers the ENACT, YIELD, and GAUGE phases of the DMS lifecycle.

## வடிவம் / Abstraction (Mind: WHAT TYPE)

**DESIGN**: Four reusable infrastructure modules (Memory, Dispatchers, Circuit Breaker, Reflection) with optional Context7 bridge for versioned documentation retrieval. Async-first architecture with graceful degradation. Core primitives imported by DMG and other ecosystem consumers.

## சங்கிலி / Chain (Body: HOW)

`pip install agentic-kit` → `SemanticMemoryStore` for embedding-based retrieval → `LoggingDispatcher` for safe execution → `CircuitBreaker` for failure handling → `ReflectionLoop` for post-action self-critique. Optional `[embeddings]`, `[http]`, `[all]` extras.
